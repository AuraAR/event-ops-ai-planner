# Event Operations AI Planner

A lightweight Python application that generates structured event operations plans using a Large Language Model API.

## Features
- Natural language event input
- AI-generated task checklist
- Staffing plan
- Equipment requirements
- Risk assessment
- Timeline overview

## Tech Stack
- Python 3
- OpenAI API
- Virtual environment (venv)

## Purpose
Built to experiment with LLM-powered workflow automation and structured planning generation.

## Example Use Case
Input:
Outdoor 3-day music festival for 5,000 attendees

## Engineering Improvements
- Structured JSON output generation
- Response latency tracking
- Timestamped output logging
- Modular function-based architecture

Output:
Structured operational breakdown including staffing, logistics, risk planning, and scheduling.# event-ops-ai-planner
